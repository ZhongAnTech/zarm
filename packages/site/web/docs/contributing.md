# 贡献 Zarm

## 安装

`git clone` 拉取代码仓库后，安装依赖

```bash
# 使用 pnpm 安装工作区依赖
$ pnpm install
```

然后, 你可以运行 `Zarm Design` 的本地站点。

```bash
# 开发 h5 端
$ pnpm dev
```

## 检查

代码风格检查。

```bash
$ pnpm lint

# ts/js 文件检查
$ pnpm lint:script

# 样式文件检查
$ pnpm lint:style
```

## 测试

运行完整的单元测试用例。

```bash
$ pnpm test
```
