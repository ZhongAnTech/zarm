## Install

use `npm`

```node
npm install @zarm-design/cli --save-dev
```

or use `yarn`

```node
yarn add @zarm-design/cli --dev
```

## Usage

```node
zarm build [options]       打包编译仓库
zarm dev [options]         运行开发环境
zarm deploy [options]      部署官网站点
zarm add [ComponentName]   新增组件模板
zarm --help                查看帮助信息
zarm --version             查看版本信息
```
